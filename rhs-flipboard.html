RHS FlipBoard · Setup
Room 27 · One-time configuration
1
Create a Google Sheet with 4 columns: Screen Line Text Speed
Each screen uses 7 rows going down. Screen = number, Line = 1-7, Text = what displays, Speed = seconds (on Line 1 row only). Each row = one full board screen. Speed is seconds (default: 8). Row 1 must be the header row.
2
Click File → Share → Publish to web. Choose Sheet1, format Comma-separated values (.csv), click Publish.
3
Copy the published CSV URL and paste it below. The board syncs automatically every 60 seconds.
